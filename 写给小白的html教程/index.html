* 你好，我是这个小镇的镇长，我的名字叫html，我们镇有两位村长，他们分别是head和body。

* 先给你介绍一下我们三个：

HTML，超文本标记语言，标准通用标记语言下的一个应用。“ 超文本”就是指页面内可以包含图片、链接，甚至音乐、程序等非文字元素。 其中“head”部提供关于网页的信息，“body”部分提供网页的 具体内容。
代码：

		            <html>
		              <head>
		                <title>浏览器上标签的内容<⁄title>
		              <⁄head>
		              <body>
		                <h1>第一个自然段<⁄h1>
		                   <p>第一个自然段的段落<⁄p>
		              <⁄body>
		            <⁄html>
标签 释意
<html> <html> 与<html> 之间的文本描述网页
<head> <head> 与 </head> 之间的文本描述网页的头部
<title> <title> 与 </title> 之间的文本是网页的名称
<body> <body> 与 </body> 之间的文本是可见的页面内容
<h1> <h1> 与 </h1> 之间的文本被显示为标题
<p> HTML 段落是通过<p> 标签进行定义的。
